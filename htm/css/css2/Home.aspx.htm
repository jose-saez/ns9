Bienvenidos a Microsoft Windows Server 2003
paso a paso

Este curso ha sido diseñado para brindar una actualización de conocimientos y habilidades sobre la nueva plataforma Microsoft Windows Server 2003.

El mismo está dirigido a todos los profesionales, de Informatica con interés en el área de Infraestructura Tecnologia con especial foco en Administración de Redes y/o Servidores y Operaciones.

También representa una posibilidad de crecer profesionalmente dentro de la carrera de Microsoft Certified Professional para Microsoft Windows Server 2003 (para mas información haga click aquí)

 

Con este curso los participantes adquirirán los conceptos y habilidades necesarias para comenzar a realizar Implementaciones y/o Migraciones de Microsoft Windows Server 2003, capacitandose también sobre las nuevas funcionalidades.


El entrenamiento  tiene como requisitos básicos los conocimientos previos de redes sobre sistemas operativos como Microsoft Windows 2000 o Microsoft Windows NT 4.0. También podrá ser aprovechado por aquellos que provengan de otras plataformas de servidores de redes.
